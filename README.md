📋 Task Management Application

A responsive and user-friendly Task Management Application built using HTML, CSS, and JavaScript. This project allows users to efficiently manage their daily tasks by adding, completing, and deleting tasks through an intuitive interface.

✨ Features
➕ Add new tasks instantly
✅ Mark tasks as completed
🗑️ Delete unwanted tasks
📱 Fully responsive design for desktop and mobile devices
🎨 Clean and modern user interface
⚡ Dynamic task handling using JavaScript and DOM manipulation
🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and responsive design
JavaScript (ES6) – Application logic and interactivity
🎯 Project Objective

The purpose of this project is to understand the fundamentals of front-end web development, including DOM manipulation, event handling, and dynamic data management. It serves as a beginner-friendly project for learning how interactive web applications are built.

🚀 Getting Started
Clone the repository.
Open the project folder in VS Code.
Run index.html using the Live Server extension or any web browser.

This project demonstrates the core concepts of building a task management system and can be further enhanced by adding features such as user authentication, local storage, task categories, due dates, and backend integration.
